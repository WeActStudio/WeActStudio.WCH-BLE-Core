## 如何下载程序
方法
1. 安装`Tools`目录下的`WCHISPTool_Setup.exe`
2. 安装`Tools`目录下的`CH372DRV.EXE`或者从`http://www.wch.cn/downloads/CH372DRV_EXE.html`下载
3. 按住BOOT键然后用USB数据线连接电脑，释放BOOT键
4. 打开 `WCHISPTool` && 找到 `CH57x Series` 或 `CH58x Series`
